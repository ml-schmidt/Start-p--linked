# Project Goal
Build a high-end, interactive presentation website (slide deck) for a workshop named "LinkedIn Masterclass".
The target audience is Master's students in "Materiel Kultur" (Humanities/Anthropology).
The content combines Magnus Løv Schmidt's personal growth story with strategic advice from the "Djøf LinkedIn Guide" (Rasmus Gullaksen).

# Tech Stack (Mandatory)
- Framework: Next.js 14+ (App Router).
- Styling: Tailwind CSS.
- Animations: Framer Motion (Crucial for smooth slide transitions).
- Icons: Lucide React.
- Font: Inter or Geist Sans.

# Design System ("Dark SaaS" Aesthetic)
1. Background: Deep dark slate (#020617 or #0F172A).
2. Cards (Bento Grid Style):
   - Background: Dark semi-transparent (bg-slate-900/60).
   - Border: Thin borders (border-slate-800).
   - Text: White headers, Slate-400 body text.
   - Accents: Use Electric Blue (#3B82F6) for primary actions and Emerald Green (#10B981) for "Do this" examples.
3. Navigation:
   - Bottom: A centered row of dots indicating progress.
   - Transitions: Slides should slide in from right to left smoothly.

- Source: Djøf Guide.

# Implementation Detail
1. Create a `slides.ts` data file containing all the text content above.
2. Create a `SlideRenderer` component that takes the slide data and renders the correct layout (TitleLayout, GridLayout, SplitLayout, etc.) based on a `type` property in the data.
3. Ensure Framer Motion handles the enter/exit animations perfectly.
